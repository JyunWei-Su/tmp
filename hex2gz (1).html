// offline utility v2.0

HEX → GZ / GPG

支援 gzip 壓縮檔 及 GPG 對稱加密 檔案還原
自動偵測格式 — 貼入 hexdump 後即時判斷

UNKNOWN
01 — hex dump 輸入
LINES: 0
HEX CHARS: 0
BYTES: 0
FORMAT: —
02 — 輸出檔名
03 — 執行
Linux 端產生方式

【GZ 壓縮】
  gzip -9 -c <file> > file.gz
  hexdump -v -e '64/1 "%02x"' -e '"\n"' file.gz

【GPG 對稱加密】
  gpg --symmetric --cipher-algo AES256 -o file.gpg <file>
  hexdump -v -e '64/1 "%02x"' -e '"\n"' file.gpg

① 複製輸出文字，貼入上方輸入框（格式自動偵測）
② 設定輸出檔名後點擊 CONVERT & DOWNLOAD
③ 解密：gpg --decrypt file.gpg > output

注意：完全在瀏覽器端執行，不需網路，不上傳任何資料_
